# Decision Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| implementation_effort | 3 | How much work to implement (lower effort = higher score) |
| correctness | 5 | Does it solve the actual problem identified? |
| maintainability | 4 | Ongoing burden — does it create toil or reduce it? |
| ecosystem_fit | 4 | How well does it fit with existing Eidos tooling (forges, registry, trilogy)? |
| risk | 3 | What can go wrong — scope creep, breaking changes, false confidence? |
